installed IDE 1.8.7 from teensyduion site

tar -xf package

./arduino---failed because of -> libgif.so.7: cannot open shared object file: No such file or directory

install font-forge and font-forge-common ---didn't use this

sudo apt-get install software-properties-common
sudo add-apt-repository ppa:fontforge/fontforge 
sudo apt-get update
sudo apt-get install fontforge-common
sudo apt-get install fontforge

install arduino from apt-get (1.0.5) -- this installs the missing libraries

sudo apt-get install arduino
sudo apt-get purge arduino

install arduino 1.8.7 provided from teensyduino site and cd to Downloads folder

cd ~/Downloads

wget https://www.pjrc.com/teensy/td_144/arduino-1.8.7-linuxaarch64.tar.xz

tar -xf arduino-1.8.7-linuxaarch64.tar.xz

cd arduino-1.8.7/

./install.sh

Now arduino IDE will be a shortcut on desktop
